Components

Page layout

Wrapper

Each page of the worksheet is wrapped in a <div> with a class of worksheet. This has styling attached to it to give it a maximum width and center it in the browser.

An <h1 class="page-title"> acts as the title of the worksheet with the <form> following it.

Sections

Sections of the form are wrapped in a <fieldset> with a <legend> which helps screenreaders to group the fields together and give them a label. The fieldset has a class of block which applies a border to the box. The worksheet__section class currently doesnt have any styling associated with it and functions only as a helpful guide in the markup.

Formatted fields

These fields use Cleave.js to apply formatting and other rules to the input entry to help guide the user.

Cleave must be applied to each individual input so the inputs are targeted with querySelectorAll by class name and then looped through with a forEach in case there are multiple inputs of any given type on the page. Then Cleave is applied to the element with the desired set of options for that input type.
