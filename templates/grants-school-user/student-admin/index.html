Student header & tabs

All of the student detail views have a header with information about the student being viewed as well as tabbed navigation to the various views pertaining to the student.

The page title uses a nested <small> element to create a subheading.

The tab navigation is based on the Bootstrap tab navigation and has the extra class student__tabs which applies custom styling specific to these views like borders and font-sizes.

The tab corresponding to the active view gets an active class on the <a class="nav-link">. Tab labels will wrap to multiple lines as needed and the font-size will adjust if the user is on a smaller screen.

Content for the view is contained in a <div class="block"> with a variant class block--student-content removing the top border so that it fits with the tab navigation. To preserve the heading heirarchy, an <h2> is used as the heading for the content but an h4 class is applied to set a more appropriate font-size.
